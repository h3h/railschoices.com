The Why, How and Who of Rails Choices

This site was formed out of a recurring desire to know what my options were when creating a new Rails app. I never knew quite where new plugins or libraries fit in the grand scheme of building a Rails app, or how they compared with existing solutions. Should I use atttachment_fu or Paperclip? Do they even provide the same functionality? Hopefully this site will supply the ongoing answers to such questions.

The site is written in YAML and HTML, using the HTML5 doctype with HTML4 elements. The source is stored on GitHub and the files are generated with Jekyll on GitHub Pages, which hosts the site by way of a DNS pointer from MediaTemple. The color scheme was inspired by the Wasabi Suicide palette on COLOURlovers. Primary page headers are rendered with the open-source Chunk font from The League of Movable Type, in those browsers that support @font-face.

This site was conceived, planned and created by Brad Fults, a web developer in San Francisco, California.
